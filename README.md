# ZhihuMoyuMac

一个简洁的 macOS 摸鱼客户端（SwiftUI）。

## 已实现（v0.1）

- 首页推荐流（未登录）
- 热搜列表
- 详情阅读（内嵌 HTML 渲染）
- 评论查看（根评论 + 子评论展开）
- 本地收藏（UserDefaults）
- 搜索过滤（本地）
- 老板键（低存在感显示）

## 运行

```bash
cd /Users/gongping/Documents/随便/ZhihuMoyuMac
swift run
```

也可以直接用 Xcode 打开 `Package.swift` 运行。
